# 爬楼梯

假设你正在爬楼梯。需要 n 阶你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

&#x20;

示例 1：

输入：n = 2 输出：2 解释：有两种方法可以爬到楼顶。

1. 1 阶 + 1 阶
2. 2 阶 示例 2：

输入：n = 3 输出：3 解释：有三种方法可以爬到楼顶。

1. 1 阶 + 1 阶 + 1 阶
2. 1 阶 + 2 阶
3. 2 阶 + 1 阶 &#x20;

提示：

1 <= n <= 45

```
func climbStairs(n int) int {

    f1 := 1 
    f2 := 2
    f3 := f1 + f2

    if n <= 2 {
        return n
    }

    for i := 3 ; i <= n ; i ++ {
        f3 = f1 + f2 
        f1 = f2
        f2 = f3 
    }
    return f3 
}
```
